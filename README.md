# Simple Chat App

ແອັບແຊັດງ່າຍໆທີ່ເຮັດດ້ວຍ React

## ຄຸນສົມບັດ

- ສົ່ງແລະຮັບຂໍ້ຄວາມ
- ບອດຕອບກັບອັດຕະໂນມັດ
- ປ່ຽນຊື່ຜູ້ໃຊ້ໄດ້
- ລຶບປະຫວັດແຊັດໄດ້
- ສະແດງເວລາຂອງຂໍ້ຄວາມ
- UI ທີ່ຕອບສະໜອງໄດ້

## ວິທີຕິດຕັ້ງ

1. Clone repository ນີ້:
```bash
git clone https://github.com/[username]/simple-chat-app.git
cd simple-chat-app
```

2. ຕິດຕັ້ງ dependencies:
```bash
npm install
```

3. ເລີ່ມແອັບ:
```bash
npm start
```

ແອັບຈະເປີດຢູ່ `http://localhost:3000`

## ການໃຊ້ງານ

- ພິມຂໍ້ຄວາມໃນຊ່ອງດ້ານລຸ່ມ
- ກົດ Enter ຫຼືປຸ່ມສົ່ງເພື່ອສົ່ງຂໍ້ຄວາມ
- ປ່ຽນຊື່ໃນຊ່ອງດ້ານເທິງ
- ກົດປຸ່ມຖັງຂີ້ເຫຍື້ອເພື່ອລຶບແຊັດ
- ບອດຈະຕອບກັບອັດຕະໂນມັດຫຼັງຈາກສົ່ງຂໍ້ຄວາມ

## ເທັກໂນໂລຍີທີ່ໃຊ້

- React 18
- Lucide React (ໄອຄອນ)
- CSS3
- JavaScript ES6+

## Scripts

- `npm start` - ເຮັດວຽກໃນໂໝດ development
- `npm run build` - ສ້າງແອັບສຳລັບ production
- `npm test` - ແລ່ນ tests
- `npm run eject` - ເອົາ React scripts ອອກ
